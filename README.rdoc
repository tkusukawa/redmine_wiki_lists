= redmine_wiki_lists

This plugin adds wiki macros that display issue lists.
(Use only if you trust all users as there is a risk of abuse.)

== Installation

* cd {RAILS_ROOT}/plugins
* git clone https://github.com/tkusukawa/redmine_wiki_lists.git
* Restart REDMINE
